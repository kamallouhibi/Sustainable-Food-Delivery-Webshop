#  FRAM – Sustainable Food Delivery WebshopA responsive, AI-integrated webshop connecting Norwegian customers with local farms.  Built by **Kamal Louhibi** as part of the *Frontend Essentials* course at **Oslo Nye Fagskole (2025).**##  Overview**FRAM** is a sustainable food delivery concept that connects customers directly to local Norwegian farms.  The webshop allows users to browse seasonal produce, learn about the farms, and contact the service — all while promoting sustainability and reuse.  The project demonstrates:- Frontend development with HTML5, CSS3, and JavaScript (ES6+)- Integration of external APIs (Leaflet + OpenAI)- Accessibility and performance optimization- Ethical and responsible AI implementation##  Features✅ **Landing Page** – Showcases the brand and sustainability mission  ✅ **Product Page** – Displays seasonal produce with interactive product cards  ✅ **Contact Page** – Allows customers to send messages via a validated form  ✅ **AI Chatbot** – OpenAI-powered assistant for customer inquiries  ✅ **Interactive Map** – Leaflet + OpenStreetMap integration showing partner farms  ✅ **Newsletter Signup** – With name/email validation  ✅ **Responsive Layout** – Adapted for desktop, tablet, and mobile  ✅ **WCAG 2.1 AA Accessibility** – Focus states, semantic structure, and ARIA labels  ##  Tech Stack**Languages & Frameworks:**  - HTML5  - CSS3 (Flexbox + Grid)  - JavaScript (ES6+)  **APIs & Tools:**  - [OpenAI API](https://platform.openai.com/) – for chatbot functionality  - [Leaflet.js](https://leafletjs.com/) + [OpenStreetMap](https://www.openstreetmap.org/) – for map integration  - Git & GitHub – for version control and project management  - Chrome DevTools + Lighthouse – for testing performance and accessibility  ##  How to Run Locally1.
